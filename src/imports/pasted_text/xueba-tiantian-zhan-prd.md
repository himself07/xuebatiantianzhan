二、产品基础信息与核心定位
2.1 基础信息
•产品名称：学霸天天战
•产品形态：微信小程序（无需下载，扫码/搜索直达，轻量化打开，适配手机端）
•目标用户：6-12岁小学1-6年级学生，核心聚焦2-5年级有数学同步练习需求、反感枯燥刷题的学生；兼顾关注产品教育属性、反对过度游戏化的学生家长
•核心学科：V1.0仅上线小学数学（同步课内知识点，覆盖口算、计算、几何、应用题），后续迭代语英学科
•商业化模式：V1.0无任何充值入口、无付费解锁、无广告，纯公益轻量化教育产品，后续迭代考虑家校联动、学习报告增值服务，绝不涉及付费养成
2.2 核心定位
以学霸乐园（个人书房）为唯一默认首页的轻量化数学教育微信小程序，严格落地“学玩分离”核心理念，个人场景主打5-8分钟碎片化专心答题，守住教育产品底色；主世界升级为校园+班级场景，作为学习后的正向奖励场景，承载轻社交、班级竞技功能，融合宠物陪伴、卡牌养成、班级集体荣誉感，实现“学完再玩、玩促学习”的正向循环，摒弃纯刷题枯燥感与纯游戏化沉迷，打造家长放心、孩子主动学的教育产品。
2.3 核心差异化优势
•场景边界清晰：纯个人学习场景+社交竞技场景彻底分离，杜绝一打开即游戏化，贴合家长教育认知
•玩法绑定学习：所有养成、竞技、资源均绑定答题行为，无纯氪金、纯挂机养成，避免养卡不学习
•低龄适配性强：操作极简、视觉温馨、零惩罚机制，贴合6-12岁孩子认知与操作能力，无挫败感
•轻社交不沉迷：班级正向社交+轻对抗，无恶意攀比、无资源掠夺，强化集体学习氛围
三、市场与用户需求分析
3.1 市场痛点
•传统答题类产品：纯文字刷题，界面枯燥，无留存钩子，孩子被动完成任务，缺乏主动学习动力
•游戏化教育产品：过度游戏化，学习场景与娱乐场景混淆，家长抵触，孩子易沉迷纯玩法忽略学习
•同类产品短板：无班级社交属性，缺乏集体荣誉感，个人单打独斗，难以形成长期使用习惯
3.2 用户核心需求
学生端需求
•不想枯燥刷题，希望学习过程有陪伴感、有乐趣、有即时反馈
•渴望和同班同学互动，有集体归属感，通过正向竞技获得成就感
•操作简单易懂，耗时短，不占用大量课余时间，答错无惩罚，无心理压力
家长端需求
•产品以学习为核心，无过度游戏化、无充值、无沉迷诱导
•能看到孩子真实学习成果，巩固课内知识点，养成良好学习习惯
•轻量化使用，无需下载APP，方便监管孩子使用时长
四、整体场景架构与核心原则
核心总原则：学玩分离，入口清晰；学习为核心，游戏化为奖励；先学后玩，杜绝沉迷；教育优先，公平无氪
4.1 双场景核心架构（唯一架构，无其他分支）
场景一：学霸乐园（个人书房）—— 默认首页，纯学习主场
•定位：产品唯一默认启动页，无任何社交、对抗、游戏化娱乐元素，纯个人学习空间，孩子打开即进入专心学习模式
•核心价值：守住家长对教育产品的核心认知，强化产品学习属性，沉淀所有基础学习功能，实现碎片化高效答题
•承载内容：每日闯关、趣味答题、错题本、学习统计、宠物小屋、抽卡中心、个人主页、主世界唯一入口
场景二：主世界（校园+班级）—— 学习奖励，社交竞技主场
•定位：学霸乐园延伸场景，非默认页，仅完成每日学习任务后可进入，属于学习后的趣味拓展奖励
•核心价值：满足孩子社交、竞技、探索需求，将娱乐欲望转化为学习动力，强化班级归属感，提升产品留存
•承载内容：校园广场、班级专属领地、学校竞技场、答题补给站，所有社交、班级对抗、集体活动均在此场景
4.2 场景跳转逻辑
启动小程序 → 直接进入学霸乐园（纯学习）→ 完成每日核心答题任务 → 解锁主世界进入权限/竞技体力 → 点击校园大门图标一键进入主世界（社交竞技）→ 点击返回键一键回到学霸乐园，无多层跳转、无复杂路径，场景边界极致清晰。
五、核心功能模块详细设计
5.1 学霸乐园（个人书房）功能详解
5.1.1 场景基础规范
•场景风格：2D平面卡通静态场景，明亮温馨，贴合校园书房氛围，元素极简（书桌、书架、宠物小屋、功能图标）
•交互方式：纯点击式互动，无摇杆、无跑跳、无自由移动，操作极简，低龄孩子一键上手
•宠物设定：宠物固定站位，仅做小幅互动，承担读题、鼓励、错题提示教学功能，无游戏化追逐打闹
5.1.2 核心学习功能（优先级最高）
•每日闯关：每日刷新，2道基础小题+1道综合大题，不重复不积压，完成即得高级口粮+大额金币，首通额外奖励，主打5分钟核心学习
•趣味答题角：碎片化单题练习，随时可做，无压力，完成得小额奖励，适配课余碎片时间
•学习成果统计：卡通图表展示每日/每周答题数、正确率、知识点掌握情况，直观呈现学习效果，支持家长查看
•错题本：自动收录错题，支持重复订正，订正完成得爱心口粮，强化知识薄弱点，无负面惩罚
5.1.3 养成功能（全绑定答题）
•宠物小屋：查看宠物状态（饱食度、亲密度、等级），仅可使用答题获取的口粮喂养，无其他获取途径
•抽卡中心：消耗答题金币抽卡，唯一卡牌获取渠道，设置保底机制，累计次数必出高稀有卡牌，保障公平性
•个人主页：展示学习成果、宠物形象、卡牌、班级归属，仅展示学习数据，不展示战力，杜绝攀比
5.1.4 主世界入口
场景内设置卡通校园大门图标，与学习功能图标并列排布，视觉显眼不突兀，点击一键跳转主世界，右上角设置返回按钮，无任何跳转门槛，未完成每日学习任务时，图标置灰，提示“完成今日答题即可进入趣味校园”。
5.2 主世界（校园+班级）功能详解
5.2.1 场景基础规范
•场景形态：2D极简卡通校园大地图，还原小学实景，无复杂地形，低开发成本，仅保留奔跑、蹦跳基础触屏操作
•核心区域：校园广场（枢纽）、班级领地、学校竞技场、答题补给站，四大区域入口清晰，无隐藏探索
•体力限制：所有竞技、进攻行为消耗体力值，体力仅通过学霸乐园答题、补给站答题获取，无购买途径
5.2.2 校园广场（核心枢纽）
主世界中转中心，连接各功能区域，支持轻量化探索，触碰场景小物件触发学习鼓励语、小额金币；设置校级学习竞技榜，仅展示答题正确率、胜率，展示学霸班级荣誉标识，提供卡牌整理、宠物状态查看功能，为竞技做前置准备。
5.2.3 班级专属领地（核心新增）
基础设定
每个班级独立卡通教室/小院领地，加入班级自动解锁，按年级在广场有序排列；通过班级集体答题积分解锁装饰（黑板报、绿植、荣誉牌），装饰越丰富，领地防御buff越强，所有装饰无氪金、纯绑定学习。
班级领地轻攻防玩法（教育优先，无恶意掠夺）
•核心规则：答题表现占70%，卡牌战力占30%，答题正确率+速度决定胜负，杜绝纯战力碾压
•进攻规则：完成每日答题获得每日进攻次数，无额外购买，匹配同年级题目，答题越好卡牌输出越高
•防守规则：成员完成碎片化答题激活守护岗，提升防御buff，获额外奖励；无在线成员时，系统模拟班级平均答题水平防守
•奖励与惩罚：胜利获班级积分+个人卡牌碎片，防守成功解锁领地装饰；平局/失败无任何惩罚，仅弹出鼓励语，保护孩子兴趣
5.2.4 学校竞技场（校级跨班对抗）
•个人跨班战：匹配同年级相近水平同学，答题+卡牌辅助比拼，胜利获校级积分，提升个人排名
•班级小队战：3-5人班级组队，全员同步答题，累计正确率定胜负，胜利为班级赢荣誉，全员得限定头像框、卡牌外观
•排名展示：仅展示学习相关数据，不展示纯战力，强化学习核心导向
5.2.5 答题补给站
主世界体力值唯一补充渠道，单题碎片化答题，题型简单，答错无惩罚，完成得体力+小额奖励，严格控制每日答题次数，避免过度学习，确保所有竞技行为均绑定学习。
5.3 基础用户模块
•登录体系：微信一键快捷登录，无需注册，选择年级、预设班级即可使用，降低操作成本
•新手引导：1分钟极简引导，宠物语音教学，教会核心操作，引导完成后解锁主世界权限
•个人主页双版展示：乐园版（学习数据）、主世界版（竞技养成成果），贴合场景定位
5.4 宠物养成模块（情感核心，留存钩子）
•基础设定：初始萌宠，支持自定义起名，仅设饱食度、亲密度、等级三大数值，易懂易操作，无饥饿掉级惩罚
•升级逻辑：饱食度+亲密度达标自动升级，分幼年期、成长期、成熟期，外观可爱变化，无暴力进化
•核心能力：乐园内读题、错题提示、学习鼓励；主世界场外助战、释放大招，无正面战斗
•情感羁绊：宠物状态随喂养情况变化，触发思念语音，唤醒孩子回归学习，形成情感牵挂
5.5 学霸卡牌模块（竞技辅助，非核心）
•卡牌设定：按稀有度S/SR/R分级，对应小学数学知识点，形象可爱无暴力，仅数学系卡牌
•获取养成：金币抽卡+碎片合成，升级升星纯靠答题积累资源，无付费捷径
•出战规则：最多上阵3张，触发学霸羁绊，战力仅为竞技辅助，不决定胜负，避免养卡不学习
六、核心玩法闭环与流程逻辑
6.1 个人学习闭环（学霸乐园）
启动进入学霸乐园 → 完成每日闯关/趣味答题（5-8分钟）→ 获取口粮、金币、体力 → 喂养宠物升级、抽卡养成卡牌 → 解锁乐园小装饰、宠物外观 → 提升学习战力与个人成就感 → 次日主动回归学习，形成纯个人正向学习循环。
6.2 学玩联动闭环（双场景）
完成乐园每日学习任务 → 解锁主世界权限+竞技体力 → 进入校园主世界 → 参与班级攻防、竞技场竞技、班级互动 → 赢取班级荣誉、限定奖励、养成资源 → 激发更强学习动力 → 返回乐园刷题提升战力，实现“学习-奖励-竞技-再学习”完整闭环。

•视觉风格：整体温馨卡通校园风，色彩明亮柔和，摒弃炫酷网游画风，宠物、卡牌、场景形象低龄化、可爱化，符合小学生审美与教育产品调性
•交互规范：乐园纯点击，主世界基础触屏跑跳，无复杂操作，按钮大、文字清晰，适配孩子操作习惯，页面跳转无延迟
•视听规范：音效轻快柔和，无刺耳噪音、无暴力特效；战斗动画为光效、星星、爱心等正向元素，无流血、无打斗画面；答错无负面音效，宠物全程温柔鼓励
八、班级社交与正向激励体系
8.1 正向社交规则
仅支持友好鼓励语、卡通表情互动，班级留言板支持学习技巧分享、老师发布任务，无恶意吐槽、无捣蛋行为、无资源掠夺，打造健康校园社交氛围。
8.2 核心激励机制（无沉迷诱导）
•情感激励：宠物陪伴、班级集体归属感，驱动主动学习
•成长激励：宠物升级、卡牌进阶、领地装饰，可视化成长收获成就感
•荣誉激励：个人排名、班级学霸荣誉、限定外观，正向好胜心引导
•学玩激励：主世界玩法作为学习奖励，强化先学后玩习惯
8.3 集体活动规划
•每周班级集体答题赛：全班共同答题积分解锁集体奖励
•每月校级班级争霸赛：跨班小队竞技，赢取全校学霸班级荣誉
九、风控规则与教育底线把控
V1.0核心风控红线：绝不触碰，全程坚守教育属性，杜绝游戏化沉迷与商业化诱导
•零惩罚机制：答错不扣分、不扣奖励、可免费重答，无失败惩罚，保护学习兴趣
•学玩严格分离：乐园无任何社交对抗元素，主世界仅学习后可进入，无反向跳转逻辑
•无商业化：无充值、无付费解锁、无广告、无付费捷径，所有资源纯答题获取
•时长管控：每日核心答题5-8分钟，主世界体力有限，单日使用时长可控，不占用过多课余时间
•对抗温和化：无恶意掠夺、无战力碾压、无负面互动，胜负仅看答题表现
•内容合规：题目同步课内知识点，无超纲、无低俗内容，符合小学教育规范
