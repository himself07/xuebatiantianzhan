## 一、核心纠正：命名绑「题型」，不绑「数字」

### 1.1 错误做法 ❌

| 做法 | 问题 |
|------|------|
| `84÷7` →「八四分七怪」 | 把算式数字硬塞进名字，难记、不好聊、像密码 |
| 按 `division_slime` 泛称「除法史莱姆」 | 孩子只记得怪物种类，不记得今天学哪类题 |

### 1.2 正确做法 ✅

**Boss 外号 = 母题类型（一类题）的童趣人设**

| 母题类型 | 今日外号 | 母题一句话 | 题型举例（可含数字） |
|----------|----------|------------|----------------------|
| 切刀问题 | **切猪肉怪** | 切几段要几刀 | 切8块要几刀？切18块要几刀？ |
| 平均分问题 | **分糖大盗** | 分东西每人几份 | 84颗糖分7人；总量÷份数 |
| 乘法应用 | **采购买笔怪** | 每组几个×买几组 | 每盒7支买8盒 |
| 两步应用题 | **零花钱巨龙** | 先求一部分再合并 | 书15元笔贵12元 |

**记忆公式（给孩子）**：

> 记住两个词：**今日怪名** + **母题一句话**  
> 例题里的数字每天可以变，怪名和题型不变。

**记忆公式（给家长/老师）**：

> 看海报认**母题类型**；孩子和同学聊**怪名**，回家还能说出「今天学的是切几段要几刀」。

## 二、双名体系（A5）

| 字段 | 示例 | 用途 |
|------|------|------|
| `dailyNickname` | 切猪肉怪 | 主展示、分享标题、同学话题 |
| `name`（系列名） | 除法史莱姆 | 立绘 IP、图鉴系列、视觉统一 |
| `topicTypeLabel` | 切刀问题 | 教研/家长一眼懂类型 |
| `motherTopicSummary` | 切几段要几刀 | 母题一句话（非算式） |
| `topicExamples` | 切8块…切18块… | Gate/结算举例、聊天素材 |

**UI 展示规范**：

- 大字：**切猪肉怪**
- 小字：除法史莱姆 · 切刀问题
- 母题行：切几段要几刀

## 三、全链路贯穿（A6）

| 触点 | 展示内容 |
|------|----------|
| Gate（B2 通缉令） | ★今日通缉令★ + 外号 + 系列名 + 母题一句话 + 例题 |
| Boss 战顶栏 | 通缉：切猪肉怪 |
| BossHeader | 外号 + 系列·类型 + 挑衅语 |
| 连击（B3） | 2连击 / 3连击·母题必杀 |
| 结算 | 记住两个词卡片 + classTalk |
| 海报 | 外号主标题 + 系列副标题 + 母题类型 |
| 分享（E6） | `我23秒击败切猪肉怪！母题：切几段要几刀，敢挑战吗？` |
| 首页 | 今日通缉：切猪肉怪 · 切几段要几刀 |

## 四、A1 与 A3 协作流程

```mermaid
flowchart LR
  teach[教研列出母题类型清单]
  a1[A1 人工起第一版外号]
  a3[A3 脚本批量生成候选]
  review[教研抽检]
  json[bossNicknames.json]
  app[小程序 mergeBossNickname]
  teach --> a1 --> a3 --> review --> json --> app
```

### A1：预置词库（当前已上线）

- 文件：`src/data/bossNicknames.json`
- 每条绑定 `topicTypeId`，可挂多个 `bossKeys`（系列 Boss）
- `cutting_problem` 已作模板条目（暂未挂 bossKeys，供扩展）

### A3：大模型离线批量

```bash
# 仅输出 Prompt（无 API Key）
python scripts/generate_boss_nicknames.py --prompts

# 调用 OpenAI 兼容 API 并写入 JSON
export OPENAI_API_KEY=sk-...
python scripts/generate_boss_nicknames.py --write
```

**Prompt 原则**（详见脚本内 `SYSTEM_PROMPT`）：

- 输入：母题类型 + 类型说明 + 例题场景
- 输出：`dailyNickname`、`motherTopicSummary`、`topicExamples`、`taunt`、`shareHook`、`classTalk`
- 禁止：数字硬编进外号、恐怖血腥、具体算式当名字

## 五、验证指标（最小闭环 1 周）

| 指标 | 目标 | 测法 |
|------|------|------|
| 外号+母题复述率 | ≥ 4/5 人能说出一对词 | 通关后 5 人访谈 |
| 分享标题点击率 | ≥ 15% | 结算页分享按钮 |
| 愿晒意愿 | 愿主动发群 ≥ 3/5 | 访谈 |
| 同学话题 | 出现怪名讨论 | 班级群观察 |

**访谈问题示例**：

1. 今天打的怪叫什么名字？
2. 这类题要怎么算？（听母题一句话）
3. 愿不愿意把海报发到群里？

---

## 六、命名红线（教研审核）

1. 外号 4～6 字，有画面、能聊天  
2. 不把 `84÷7` 这类算式拆字进名字  
3. 母题一句话不含具体答案  
4. 海报保留「母题类型 + 一句话」  
5. 不恐怖、不血腥、不谐音低俗
